# 알고리즘

## 정렬

### 삽입정렬

- 정렬되지 않은 데이터를 이미 정렬된 부분 배열의 적절한 위치에 차례로 삽입

### 병합정렬

- 정렬되지 않은 데이터를 계속 쪼개서 제일 작은단위부터 병합해나가는 방식

```c++
#include <iostream>

using namespace std;

void merge(int arr[], int left, int mid, int right)
{
    int size = right - left + 1;
    int* tmp = new int[size];

    int i = left;
    int j = mid + 1;
    int k = 0;

    while (i <= mid && j <= right)
    {
        if (arr[i] <= arr[j])
        {
            tmp[k++] = arr[i++];
        } else
        {
            tmp[k++] = arr[j++];
        }
    }

    while (i <= mid)
    {
        tmp[k++] = arr[i++];
    }

    while (j <= right)
    {
        tmp[k++] = arr[j++];
    }

    for (int idx = 0; idx < size; ++idx)
    {
        arr[left+idx] = tmp[idx];
    }

    delete[] tmp;
}

void mergeSort(int arr[], int left, int right)
{
    if (left < right)
    {
        int mid = left + (right - left)/ 2;
        mergeSort(arr, left, mid);
        mergeSort(arr, mid+1, right);
        merge(arr, left, mid, right);
    }
}

int main()
{
    int arr[] = {9, 3, 1, 5, 13, 22};
    int n = sizeof(arr)/sizeof(arr[0]);

    cout << "원본 배열 : ";
    for (int i = 0; i < n; i++)
    {
        cout << arr[i] << " ";
    }
    cout << endl;

    mergeSort(arr, 0, n-1);

    cout << "정렬된 배열: ";
    for (int i = 0; i < n; i++)
        cout << arr[i] << " ";
    cout << endl;
}
```

- N log N 으로 굉장히 효율적

### 계수정렬

- 카운트하는 식으로 동작 O(N)으로 가능
- 아주 특수한 경우에만 사용함, 음수값이 있거나, 원소값의 범위가 너무 넓으면 공간복잡도가 너무 높아져서 못씀

### 힙 정렬

- 힙은 조건을 만족하는 이진트리
- 최대힙, 최소힙이 있음
- 최대힙의 경우 부모값이 자식보다 크거나 같음

```c++
#include <iostream>

using namespace std;

void heapify(int arr[], int n, int i)
{
  int largest = i;
  int left = 2 * i + 1;
  int right = 2 * i + 2;
  if (left < n && arr[left] > arr[largest])
  {
    largest = left;
  }
  if (right < n && arr[right] > arr[largest])
  {
    largest = right;
  }
  if (largest != i)
  {
    swap(arr[i], arr[largest]);
    heapify(arr, n, largest);
  }
}

void heapSort(int arr[], int n)
{
  for (int i = n / 2 - 1; i >= 0; i--)
  {
    heapify(arr, n, i);
  }

  for (int i = n - 1; i >= 0; i--)
  {
    swap(arr[0], arr[i]);
    heapify(arr, i, 0);
  }
}

int main()
{
  int arr[] = {4, 10, 3, 5, 1};
  int n = sizeof(arr) / sizeof(arr[0]);

  heapSort(arr, n);

  for (int i = 0; i < n; i++)
  {
    cout << arr[i] << " ";
  }
  cout << endl;
}
```
