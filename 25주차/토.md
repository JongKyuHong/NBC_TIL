## C++ 문자열 STL

### 특정 값이 문자열에 포함되어 있는지 확인, find, contains

```c++
#include <string>

// C++23부터는 contains로 판단 가능
std::string str = "Hello C++";
str.contains("C++") // true

// C++20이하 find + npos로

std::string str2 = "Hello, C++";

if (str2.find(C++) != std::string::npos){
    // ...
}
```

### 특정 구간값 추출하기 substr

```c++
#include <string>

int main(){
    std::string str = "Hello, C++ World!";

    // 3번째 인덱스부터 3개 추출
    std::string sub = std.substr(7, 3); // C++
}
```
