### 배운 점

1. C++에서 tolower는 cctype을 선언하고나서 tolower(c); 이런식으로 사용

```c++
#include <iostream>
#include <string>
#include <cctype> // tolower

int main() {
    std::string s = "Hello WORLD!";

    // 원본 수정을 위해 &사용
    for (char& c : s) {
        c = std::tolower(c);
    }

    std::cout << s << std::endl; // 출력: hello world!
    return 0;
}
```

- 위는 한글자씩 변환할때의 예시
- algorithm헤더의 transform을 사용하면 한번에 변환할 수 있다.

```c++
#include <iostream>
#include <string>
#include <algorithm> // transform
#include <cctype>    // tolower

int main() {
    std::string s = "Hello WORLD!";

    // 인자: 시작점, 끝점, 결과를 덮어쓸 시작점, 적용할 함수
    std::transform(s.begin(), s.end(), s.begin(), ::tolower);

    std::cout << s << std::endl; // 출력: hello world!
    return 0;
}
```

- transform 시작점, 끝점, 결과를 덮어씌우기 시작할 시작점, 적용할 함수

여기까지가 현 구조를 만드는데 사용한 부분이다. 하지만 캐시에 들어온순서를 Value로 판단을 해야해서 캐시 미스가 난경우 map을 전부 순회해야 한다.

2. 순서를 담당하는 list를 사용해보자
