# 인게임 상호작용 구현

## Enhanced Input System

### Axis

- Digit(boolean) bool x
- Axis1D(float) float x
- Axis2D(Vector2D) float x, y;
- Axis3D(Vector) float x, y, z;

- 키보드 입력 : Digit 누르거나(1), 안누르거나(0)
- 마우스 좌표 : Axis2D 마우스 좌표(screen space, 화면 공간) x, y

키보드 입력 : 움직임을 키보드 입력으로 구현하지 않는가? W, S, D, A
키보드 입력은 Digit => Axis2D 입력을 받음
앞뒤를 x축, 오왼을 y축에 놓을경우
W의 경우 x(1), y(0) => Modifier가 필요없음
S의 경우 x(-1), y(0) => Modifier로 음수 전환 필요 -> Negate 필요
D의 경우 x(0), y(1) => Modifier로 x축과 y축의 전환 필요 => swizzle(YXZ)
A의 경우 x(0), y(-1) => Modifier와 x축과 y축의 전환 필요 -> swizzle(yxz), 음수 전환 필요 -> Negate

오왼이라고 한 이유 Forward Vector, Right Vector를 쓰기 때문에

## 깨알 팁

- 콜백을 당하는 함수, 남이요청하는 함수는 표시를 위해 On을 붙인다 (많이 본 패턴)
- 2번이상 불리면 무조건 따로 변수로 빼자, 정리를 잘하자

# 알고리즘

## 개선 전

```c++
// https://school.programmers.co.kr/learn/courses/30/lessons/17681?language=cpp

#include <string>
#include <vector>
#include <iostream>

using namespace std;

string makeBinary(int n, int value){
    string result = "";
    while (value > 0){
        int tmp = value % 2;
        result = to_string(tmp) + result;
        value /= 2;
    }

    while (result.length() < n){
        result = '0' + result;
    }

    return result;
}

vector<string> solution(int n, vector<int> arr1, vector<int> arr2) {
    vector<string> answer;

    // 각행의 값을 이진수로 변환, 이진수의 길이는 n
    // 0이면 공백 1이면 벽 >> 둘다 1이면 벽으로 쳐준다. 비트연산자로 하면 편하게 할 수 있을거같긴함

    for (int i = 0; i < n; i++){
        // 둘다 변환?
        string arr1BinaryStr = makeBinary(n, arr1[i]);
        string arr2BinaryStr = makeBinary(n, arr2[i]);
        string result = "";
        // cout << "arr1 : " << arr1BinaryStr << "\n" << "arr2 : " << arr2BinaryStr << "\n";
        for (int j = 0; j < n; j++){
            if (arr1BinaryStr[j] == '0' && arr2BinaryStr[j] == '0'){
                result += ' ';
            } else {
                result += '#';
            }
        }
        answer.push_back(result);
    }

    return answer;
}
```

- 이진수 변환 함수를 만들어서 문제를 풀었다.
- 간단한 문제였기때문에 이렇게도 풀 수 있지만 문제 구조상 누가봐도 비트연산으로 계산하는것 같았다.

## 개선 후

```c++
// https://school.programmers.co.kr/learn/courses/30/lessons/17681?language=cpp

#include <string>
#include <vector>
#include <iostream>

using namespace std;

vector<string> solution(int n, vector<int> arr1, vector<int> arr2) {
    vector<string> answer;

    // 각행의 값을 이진수로 변환, 이진수의 길이는 n
    // 0이면 공백 1이면 벽 >> 둘다 1이면 벽으로 쳐준다. 비트연산자로 하면 편하게 할 수 있을거같긴함

    for (int i = 0; i < n; i++){
        int combined = arr1[i] | arr2[i];
        string result = "";
        for (int j = 0; j < n; j++) {
            if (combined % 2 == 1) result = "#" + result;
            else result = " " + result;
            combined /= 2;
        }
        answer.push_back(result);
    }

    return answer;
}
```

- |를 통해 이진수 비교를 하고 combined에 2를 나누어서 자릿수 비교를 해주었다.
