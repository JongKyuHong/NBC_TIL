# 알고리즘

## 최소 신장 트리

신장트리 : 어떤 그래프 내의 모든 정점을 포함하면서 정점들끼리 서로 연결되어있고 사이클이 없는 그래프

- 최소 : 간선 가중치 합이 최소
- 신장 : 모든 정점이 하나도 빠짐없이 연결되어있어야 함
- 트리 : 사이클이 없어야 한다.
