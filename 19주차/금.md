# 알고리즘

## 레드-블랙 트리

자가 균형 이진 탐색트리로서 데이터 삽입, 삭제 시 트리의 높이를 일정 수준으로 유지하여 최악의 경우에도 탐색,삽입,삭제에 O(log N)의 시간복잡도를 보장하는 강력한 자료구조
std::map, std::set 등이 내부적으로 레드-블랙 트리를 사용한다.

### 규칙

1. 모든 노드는 레드 혹은 블랙 노드
2. 루트는 항상 블랙
3. 모든 리프(NIL)는 항상 블랙 (실제 데이터가 없는 끝노드)
4. 레드 노드의 자식은 반드시 블랙 (레드가 연속 2번 나올 수 없음)
5. 임의 노드에서 리프노드까지 가는 모든 경로에는 동일한 개수의 블랙 노드가 있다. (Black - Height)가 같다.

균형 맞추는 법

1. 회전 : 노드들의 위치를 좌/우로 바꾸어 트리 높이 조절
2. 색상 변경 : 부모나 삼촌 노드의 색을 바꿔 4번을 해결
